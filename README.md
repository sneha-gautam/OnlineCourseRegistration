# Online Course Registration System

This is a simple web-based project where users can register, login and enroll in different tech courses.

## What this project does
- Users can create an account and login
- View available courses
- Enroll in courses
- Prevents enrolling in same course multiple times
- Seats get reduced after enrollment
- Admin can see all student enrollments

## Tech used
- Java (Servlet + JSP)
- MySQL
- JDBC
- Apache Tomcat

## How it works
- Frontend is made using JSP
- Backend logic is handled using Servlets
- Database is connected using JDBC
- Data is stored in MySQL

## Note
To run this project:
- Setup MySQL database
- Import tables
- Add MySQL connector
- Run using Tomcat server
